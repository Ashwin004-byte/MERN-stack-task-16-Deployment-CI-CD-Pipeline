# 🚀 Full Stack Web Application (Task 16 - Deployment & CI/CD)

This is a full-stack web application developed as part of internship Task 16. The project demonstrates development, deployment, and CI/CD pipeline workflow using modern tools.

---

## 📌 Project Overview

The application includes:
- Frontend built with React
- Backend API using Node.js / Express
- Database integration (if used)
- Deployment using Vercel (frontend) and Render/AWS (backend)
- GitHub for version control and CI/CD pipeline

---

## 🛠️ Tech Stack

### Frontend:
- React.js
- HTML, CSS, JavaScript
- Axios / Fetch API

### Backend:
- Node.js
- Express.js

### Tools & Platforms:
- Git & GitHub
- Vercel (Frontend Deployment)
- Render / AWS (Backend Deployment)
